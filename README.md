# 📊 Sales Performance Analysis

## 📌 Project Overview
This project focuses on analyzing sales and profit data to understand business trends, revenue patterns, and performance gaps. The goal is to understand which products, regions, and customer segments contribute most to profitability and where improvements can be made.

---

## 🎯 Objective
Analyze sales and profit data to understand business trends and revenue patterns so that overall profitability can be improved.

---

## ❓ Key Analytical Questions
1. Which product category generates the highest sales?
2. Which region contributes the most profit?
3. Are there categories with high sales but low profit?
4. Which customer segment is the most profitable?
5. How do sales and profit trends change over time?

---

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Jupyter Notebook

---

## 📂 Dataset Source
Dataset obtained from Kaggle (Superstore Sales Dataset).  
Used for educational and portfolio purposes only.
